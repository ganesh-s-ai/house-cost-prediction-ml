# house-cost-prediction-ml
# Machine Learning Based House Construction Cost Estimation

This project uses a Random Forest Regression model to predict residential construction costs based on structural parameters such as square feet, number of rooms, bathrooms, kitchen, sitout, and floors.

## Technologies Used
- Python
- Scikit-learn
- Flask
- HTML
- CSS
- JavaScript

## Features
- Real-time cost prediction
- Web-based interface
- Machine learning model integration

## How to Run
1. Install dependencies using `pip install -r requirements.txt`
2. Run backend using `python app.py`
3. Open frontend in browser
